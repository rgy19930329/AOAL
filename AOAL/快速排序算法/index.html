算法思路

快速排序思想的关键在于递归的运用。

每次排序都将目标数组分成三个部分：位于数组中间位置的数middle，比middle值小的数组leftArray，比middle值大的数rightArray。

然后分别将leftArray、rightArray当作目标数组进行下一次排序，也就是重复上面的步骤，直到每个数组的长度小于或等于1则停止。

时间复杂度：

平均：O(nlog2n)

最好：O(nlog2n)

最坏：O(n2)

辅助存储：O(nlog2n)

稳定性：不稳定

Array.prototype.quickSort = function(){
	if(this.length <= 1){
        return this;
    }else{
    	var leftArray = [],
		rightArray = [],
		index = Math.floor(this.length / 2),
		middleValue = this.splice(index, 1);

		leftArray = this.filter(function(item){
			return item <= middleValue;
		});

		rightArray = this.filter(function(item){
			return item > middleValue;
		});

		return leftArray.quickSort().concat(middleValue, rightArray.quickSort());
    }
}
