AOAL / 直接插入排序算法 /

算法思路

将数据整体分为两个部分，一个是有序表order，一个是无序表unorder。

直接插入排序需要做的是每次取出无序表unorder中的第一个数据，将它插入到有序表order中适当的位置即可。

值得注意的是上面描述的插入有序表中适当的位置该如何实现？重点在“适当”二字。

时间复杂度：

平均：O(n2)

最好：O(n)

最坏：O(n2)

辅助存储：O(1)

稳定性：稳定

JavaScript C/C++ Java Python 
Array.prototype.sort = function(){
	var len = this.length;
	var order = this.slice(0, 1);
	var unorder = this.slice(1);
	for(var i = 1; i < len; i++){
		var target = unorder.shift();
		for(var j = 0; j < order.length; j++){
			if(target <= order[j]){
				break;
			}
		}
		order.splice(j, 0, target);
	}
	return order;
}
