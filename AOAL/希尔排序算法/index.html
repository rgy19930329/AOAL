算法思路

希尔排序(Shell Sort)是插入排序的一种，实质上是一种分组插入方法。

希尔排序是把记录按下标的一定增量分组，对每组使用直接插入排序算法排序；随着增量逐渐减少，每组包含的关键词越来越多，当增量减至1时，整个序列恰被分成一组，算法便终止。说简单点，排序的过程就是 分组+直接插入排序。

时间复杂度：

平均：O(n1.3)

最好：O(n)

最坏：O(n2)

辅助存储：O(1)

稳定性：不稳定

Array.prototype.sort = function(){
	var i, j, k, temp, len = this.length;
	var d = len;
	while(true){
		d = Math.floor(d / 2);
		for(k = 0; k < d; k++){
			for(i = k + d; i < len; i = i + d){
				temp = this[i];
				for(j = i - d; j >= 0 && temp < this[j]; j = j - d){
					this[j+d] = this[j];
				}
				this[j+d] = temp;
			}
		}
		if(d == 1){
			break;
		}
	}
	return this;
}
