链表

链表是一种物理存储单元上非连续、非顺序的存储结构，数据元素的逻辑顺序是通过链表中的指针链接次序实现的。链表由一系列结点（链表中每一个元素称为结点）组成，结点可以在运行时动态生成。

每个结点包括两个部分：一个是存储数据元素的数据域，另一个是存储下一个结点地址的指针域。

JavaScript C/C++ Java Python 
function LNode(data){
	this.data = data;
	this.next = null;
}

// 插入节点s
s.next = p.next;
p.next = s;

// 删除节点s
p.next = p.next.next;
					
typedef struct LNode {
	ElemType data;
	struct LNode *next;
}LNode, *LinkList;

// 插入节点s
s->next = p->next;
p->next = s;

// 删除节点s
p->next = p->next->next;
