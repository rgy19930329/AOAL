算法思路

桶排序是一种典型的以空间换时间的排序方法，可以说是最快的一种排序方法，大多数情况比快排还快，若没有重复数据的情况下，它的整个过程是线性的。最重要的是桶排序必须要知道数据范围！

桶排序有许多限制，比如只能针对整型数据排序，不能对负数和浮点数进行排序等。

不过这些困难都是可以克服的，比如要对负数排序，可以先对每项加一个比较大的数，排序完成后再减去；同理，如果要对浮点数进行排序，可以先对每项乘以一个比较大的数，排序完成后再除去。

时间复杂度：

最好：O(n)

最坏：O(n2)

辅助存储：O(n)

稳定性：稳定

Array.prototype.sort = function(){
	var arr = [];
	for(var i = 0, len = this.length; i < len; i++){
		var index = this[i];
		if(arr[index] == undefined){
			arr[index] = 1;
		}else{
			arr[index]++;
		}
	}

	var res = [];
	for(var i = 0; i < arr.length; i++){
		if(arr[i] == undefined){
			continue;
		}
		for(var k = 0; k < arr[i]; k++){
			res.push(i);
		}
	}
	return res;
}
