算法思路

归并排序的实现同样借助了递归的思想，每次将目标序列从中间一分为二，直到长度为1。然后再进行归并处理，重点在这个归并函数的实现。

所谓的归并，就是将两个均已排好序的序列进行合并，并且要求在合并后的序列依旧有序。归并函数的基本形式 function merge(left, right){}

时间复杂度：

平均：O(nlog2n)

最好：O(nlog2n)

最坏：O(nlog2n)

辅助存储：O(n)

稳定性：稳定

Array.prototype.sort = function(){
	if (this.length <= 1) {
        return this;
    }

    var middle = Math.floor(this.length / 2),
        left = this.slice(0, middle),
        right = this.slice(middle),
        params = merge(left.sort2(), right.sort2());

    params.unshift(0, this.length);
    this.splice.apply(this, params);

    return this;

    function merge(left, right) {
        var result = [],
            leftIndex = 0,
            rightIndex = 0;

        while (leftIndex < left.length && rightIndex < right.length) {
            if (left[leftIndex] < right[rightIndex]) {
                result.push(left[leftIndex++]);
            } else {
                result.push(right[rightIndex++]);
            }
        }
        return result.concat(left.slice(leftIndex)).concat(right.slice(rightIndex));
    }
}
