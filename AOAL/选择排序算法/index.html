算法思路

将数据整体分为两个部分，一个是有序表order，一个是无序表unorder。

选择排序需要做的是每次取出无序表unorder中最小的数据，将它插入到有序表order的末端。

重点在于从无序表unorder中选择出最小值。这也是它叫为什么叫选择排序的原因。

JavaScript C/C++ Java Python 
Array.prototype.sort = function(){
	var len = this.length;
	var res = [];
	for(var i = 0; i < len; i++){
		var min = this[0];
		var index = 0;
		for(var j = 0; j < this.length; j++){
			if(this[j] < min){
				min = this[j];
				index = j;
			}
		}
		var temp = this.splice(index, 1);
		res.push(temp[0]);
	}
	return res;
}
