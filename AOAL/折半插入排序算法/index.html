算法思路

折半插入排序算法与直接插入排序的思想基本一致，改良的仅仅是选择插入点的方式不同而已。

大家知道，如果不是从一组随机的序列中查找，而是从一组排好序的序列里找出某个元素的位置，折半查找是个很好的方法。

同样的道理，由于我们的order已经是一个有序数组，那么寻找插入点就可以使用折半查找的方式，可以提高排序的整体效率。

JavaScript C/C++ Java Python 
Array.prototype.sort = function(){
	var len = this.length;
	var order = this.slice(0, 1);
	var unorder = this.slice(1);
	for(var i = 1; i < len; i++){
		var target = unorder.shift();
		var low = 0, high = order.length - 1;
		while(low <= high){
			var mid = Math.floor((low + high) / 2);
			if(target < order[mid]){
				high = mid - 1;
			}else{
				low = mid + 1;
			}
		}
		order.splice(low, 0, target);
	}
	return order;
}
