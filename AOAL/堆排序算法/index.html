算法思路

利用大顶堆(小顶堆)堆顶记录的是最大关键字(最小关键字)这一特性，使得每次从无序中选择最大记录(最小记录)变得简单。

堆排序将所有的待排序数据分为两部分，无序区和有序区。无序区也就是前面的最大堆数据，有序区是每次将堆顶元素放到最后排列而成的序列。每一次堆排序过程都是有序区元素个数增加，无序区元素个数减少的过程。当无序区元素个数为1时，堆排序就完成了。

本质上讲，堆排序是一种选择排序，每次都选择堆中最大的元素进行排序。只不过堆排序选择元素的方法更为先进，时间复杂度更低，效率更高。

构造初始堆，需要从最后一个非叶节点开始调整！！

时间复杂度：

平均：O(nlog2n)

辅助存储：O(1)

稳定性：不稳定

Array.prototype.sort = function(){
	var len = this.length;
	heapBuild(this, len);

	var count = 0;
	for(var i = len; i > 1; i--){
		swap(this, i - 1, 0);
		heapAdjust(this, 0, i - 1);
		count++;
	}

	return this;

	// 交换
	function swap(arr, index1, index2){
		var temp = arr[index1];
		arr[index1] = arr[index2];
		arr[index2] = temp;
	}

	// 调整
	function heapAdjust(arr, i, len){
		var left = 2 * i + 1, 
			right = 2 * i + 2;
		var max = i;
		if(i <= Math.floor(len / 2)){
			if(left < len && arr[max] < arr[left]){
				max = left;
			}
			if(right < len && arr[max] < arr[right]){
				max = right;
			}
			if(i != max){
				swap(arr, max, i);
				heapAdjust(arr, max, len);
			}
		}
	}

	//创建大顶堆
	function heapBuild(arr, len){
		var begin = Math.floor(len / 2) - 1;
		for(var i = begin; i >= 0; i--){
			heapAdjust(arr, i, len);
		}
	}
}
